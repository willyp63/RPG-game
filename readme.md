## What it is?
2D platform, rpg game using PIXI.js

### How to Start
- `npm i` - Install dependencies
- `npm run build:watch` - Build and watch files
- `npm start` - Start the dev server
- navigate to `http://localhost:8080/`

### All Commands
- `npm i` - Install dependencies
- `npm run build` - Build once
- `npm run build:watch` - Build and watch files
- `npm start` - Start the dev server
